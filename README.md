# Computer Networks (2ICSC302) - Course Repository
This repository contains **study materials, notes, lab implementations, assignments, and projects** for the **Computer Networks** course (Course Code: 2ICSC302 / SC302).

## 🎯 Purpose of this Repository

To serve as a **complete academic companion** for students taking the Computer Networks course, providing:
- Well-organized study material
- Practical lab implementations
- Exam preparation resources
- Real-world protocol understanding through code and simulations

---

## 📋 Course Overview

**Course Name:** COMPUTER NETWORKS  
**Category:** Professional Core  
**Credits:** 3 (or as per syllabus)  

### Course Objectives
- Understand the layered architecture of computer networks.
- Learn fundamental protocols at each layer of the TCP/IP and OSI models.
- Gain practical experience with network configuration, routing, and protocol implementation.

### Course Learning Outcomes (CLOs)
- Define the layered network architecture.
- Identify routing algorithms and error detection/correction mechanisms.
- Apply knowledge of network protocols in real-world scenarios.
- Analyze and troubleshoot network communication issues.

## 📚 Syllabus Units

### Unit 1: Introduction to Networks (15 Hours)
- Network Types, LAN, PAN, WAN, MAN
- OSI & TCP/IP Reference Models
- Network Devices (Hubs, Switches, Routers, etc.)
- Physical Layer: Transmission Media, Topology

### Unit 2: Addressing (15 Hours)
- IP Addressing (IPv4 & IPv6)
- Subnetting & Supernetting
- Classless Addressing
- Static & Dynamic Routing

### Unit 3: Routing Protocols (15 Hours)
- Distance Vector & Link State Routing
- RIP, OSPF, BGP
- Unicast, Multicast & Broadcast Routing

### Unit 4: Medium Access Control (15 Hours)
- ALOHA, CSMA/CD, CSMA/CA
- Ethernet, Token Ring
- Flow Control: Stop-and-Wait, Sliding Window
- Error Detection & Correction: Parity, Checksum, CRC, Hamming Codes
- Data Link Protocols: HDLC, PPP

**Labs:**
- Lab 10: Multi-Area OSPF Implementation
- Lab 11: PPP Configuration
- Lab 12: HDLC Configuration

### Unit 5: Transport and Application Layer Protocols (15 Hours)
- UDP & TCP
- Port Numbers
- Application Protocols: HTTP, FTP, SMTP, DNS, Telnet

**Labs:**
- Lab 13: BGP Implementation
- Lab 14: EIGRP Implementation
- Lab 15: Telnet Configuration
